# n1401
Trying to write a modern assembler for the 1401

# assembler
``n1401 main.S -o main.o``
## options

| -   | desc |
| --- | ---- |
| -h  | help menu |
| -v  | version |
| -o  | specify an output file |
| -g  | specify a format: (simhtape) default simhtape |
